# macro-financial-analysis

# Overview
Econometric analysis of the relationship between the US dollar and the trade balance using VAR/VECM models on time series data.

This project investigates the transmission mechanisms between FX and bond markets through the lens of USD dynamics and trade balance data. It applies Vector Autoregression (VAR) and Vector Error Correction Model (VECM) frameworks, and interprets results via Impulse Response Functions (IRF).

# Methods
VAR / VECM : modelling interdependencies between time series \
Impulse Response Functions (IRF) : analysing shock transmission across variables
Stationarity tests : ADF, KPSS
Cointegration tests : Johansen procedure
